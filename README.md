# GenAI Secretary: Email Summarization & Priority Assistant

## Overview

GenAI Secretary is a Generative AI-powered email assistant designed to improve productivity by automating common email management tasks. The system leverages Large Language Models (LLMs) to summarize emails, classify message priority, generate context-aware responses, and evaluate draft quality for clarity and tone.

The goal of the project is to reduce time spent managing email communications while helping users focus on high-priority tasks.

---

## Features

* Automatic email summarization
* Email priority classification
* Context-aware response generation
* Draft evaluation and feedback
* Prompt-engineered workflows for LLM interactions
* Productivity-focused email management automation
